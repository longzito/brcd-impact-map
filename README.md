# Bitter Root RC&D Impact Map

An interactive, embeddable impact map built with **Leaflet.js** and **OpenStreetMap**.

## Features
- Program category filters
- Marker clustering
- Project story panels
- Free & open‑source
- Wix‑compatible via iframe

## Deploy
### GitHub Pages
1. Push repo to GitHub
2. Settings → Pages → Deploy from main branch

### Wix Embed
Embed URL using an HTML iframe element.

## Update Data
Edit `/data/projects.geojson` to add or update projects.
